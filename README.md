## Hi, I'm Raphaël.

Je suis québécois (I'm from Québec (that's in Canada)). I'm currently studying computer science and learning about whatever happens to interest me at the moment.

I know a couple languages, but my gotos are Rust and TypeScript, and sometimes C#. I prefer static typing over dynamic, and functional programming over object-oriented. I'm mainly into backend development, but I also like learning about networking, programming languages and operating systems. I don't like frontend and UI, and while I wrote most of the code for this website to make it somewhat unique, I hated every second of it.

I love open-source and try to make all the code I can freely available to read and use for others ([including this website](https://github.com/raftario/raftario.github.io)). I have a couple small open-source projects of my own, and try to contribute to other, larger ones as often as I can.

Apart from coding, I like listening to music ([here's my last.fm profile](https://last.fm/user/bobwinibago)), playing music, skiing, mountain biking, playing games, and a lot of other things that wouldn't really improve this list.
